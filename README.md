# Netflix Clone

## Description

Front end only Netflix clone which fetches data from TMDB API to generate images.

### Prerequisites

Link to software that is required to install the app.

- [Firebase](https://firebase.google.com/)

## Built With

- React
- React-Youtube
- React-Movietrailer
- TMDB API
